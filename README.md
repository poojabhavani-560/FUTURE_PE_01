# AI Website Copy Generator for Local Businesses

## 📌 Project Overview
This project demonstrates how **Prompt Engineering + AI tools** can be used to generate high-quality, conversion-focused website copy for local businesses.

Many small businesses struggle with unclear website content, weak value propositions, and poor calls-to-action. This project solves that problem by creating a **structured prompt system** that generates ready-to-use website content.

---

## 🏢 Business Chosen
**FreshGlow Beauty Salon**  
Location: Eluru  
Type: Beauty & Grooming Salon

---

## 🎯 Objective
- Generate clear and engaging homepage copy  
- Create detailed service descriptions  
- Write strong call-to-action (CTA) sections  
- Adapt tone based on business type  
- Produce content ready for real-world websites  

---

## 🛠️ Tools Used
- ChatGPT (for content generation)  
- HTML, CSS, JavaScript (for website implementation)
